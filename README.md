## 📌 简介

BDS Manager 是一个使用 PyQt5 构建的图形化管理工具，专为 **Minecraft Bedrock 专用服务器（BDS）** 设计。它整合了服务器启动/停止、资源包/行为包管理、可视化配置编辑、自动备份、系统资源监控、内网穿透隧道、版本升级等常用功能，让你无需敲命令即可轻松运维基岩版服务器。

无论你是新手服主还是资深管理员，BDS Manager 都能让你更高效、更省心地管理你的 BDS 服务器。

## ✨ 主要特性

### 🖥️ 控制台
- 一键启动/停止服务器，实时显示彩色日志输出（自动高亮错误、警告、玩家事件等）
- 支持向服务器发送命令（`stop`、`list`、`op` 等）
- 带有时间戳的控制台输出，方便日志追溯

### 📦 资源包 / 行为包管理
- 图形化添加/移除资源包和行为包
- 自动读取 `manifest.json`，显示 UUID、版本、依赖等信息
- 双击或右键查看包的详细信息（名称、描述、作者、模块、依赖等）
- **手动激活/注销**包到当前世界（无需修改 json 文件）
- 自动识别已激活状态，并显示在列表中

### ⚙️ 可视化配置编辑
- 以表单形式编辑 `server.properties`，每个字段带中文提示
- 支持常用设置（游戏模式、难度、端口、最大玩家、白名单、正版验证等）
- 内置 **端口检测与更换** 工具，检测 UDP 端口占用并自动推荐可用端口
- 直接编辑 `allowlist.json`、`permissions.json`、`packetlimitconfig.json`（带 JSON 语法校验）

### 🌍 世界管理
- 查看当前世界名称、种子、难度
- 一键 **备份** 当前世界（自动压缩为 ZIP）
- **还原** 备份（支持从列表中选择历史备份）
- 快速修改世界难度（需重启服务器生效）

### 📊 系统资源监视
- 实时显示 CPU、内存、网络、磁盘使用率（进度条 + 数值）
- 绘制 **CPU 与内存使用率历史折线图**（最近 60 个点，可自定义更新频率）
- 图表支持深色/浅色主题自适应

### 🚇 隧道 (ChmlFrp 内网穿透)
- 集成 ChmlFrp 隧道管理
- 设置 `frpc.exe` 路径，编辑 `frpc.ini` 配置文件
- 启动/停止隧道，实时查看隧道输出日志
- 支持快速打开 frpc 所在目录

### 🔄 版本升级
- 自动检测官方最新稳定版/预览版（通过 HEAD 请求探测，不依赖第三方 API）
- 下载最新版本压缩包，并 **一键完成升级**
- 升级前自动备份关键文件（worlds、配置、白名单、资源包、行为包等），可自定义是否备份
- 支持手动指定版本号直接下载安装

### 🎨 主题切换
- 内置 **深色** / **浅色** 主题
- 支持 **自定义颜色**（背景、文字、强调色、按钮等）
- 主题设置自动保存

### 🛡️ 其他实用功能
- **系统托盘图标**：最小化到托盘，双击恢复，后台运行不打扰
- **文件系统监控**：外部修改资源包或配置文件时自动刷新界面（防抖）
- **配置持久化**：自动保存服务器路径、备份间隔、主题等设置
- **多线程优化**：所有耗时操作（备份、还原、下载、解压）均在后台线程执行，界面不卡顿
- **脚本与服务器目录分离**：可随意指定服务器所在路径，方便多实例管理

## 🚀 快速开始

### 环境要求
- Python 3.8 或更高版本
- Windows

### 安装依赖

1. 克隆仓库：
```bash
git clone https://github.com/yourusername/bds-manager.git
cd bds-manager
```

2. 安装所需 Python 包：
```bash
pip install -r requirements.txt
```

> 如果需要折线图功能，请额外安装 `matplotlib`：
> ```bash
> pip install matplotlib
> ```

### 运行

```bash
python bds_manager.py
```

首次启动会在脚本同目录下创建 `Server` 文件夹作为默认服务器目录。你也可以通过「设置」更改服务器路径。

### 配置文件

程序会自动生成 `bds_manager_config.json`，其中保存了服务器路径、主题、备份间隔等设置。

## 🛠️ 使用说明

### 启动服务器
1. 确保 `server.properties` 中的 `level-name` 与 `worlds/` 下的世界名称一致。
2. 点击「启动服务器」按钮，控制台会实时输出服务器日志。
3. 在下方输入框发送命令（如 `stop` 停止服务器）。

### 添加资源包/行为包
1. 切换到「资源包/行为包」标签页。
2. 点击「添加资源包」或「添加行为包」，选择包含 `manifest.json` 的文件夹。
3. 程序会自动复制包到服务器对应目录，并尝试注册到当前世界（若服务器未运行）。

### 修改配置
- 在「配置」标签页中编辑 `server.properties` 的各项参数，点击「保存」即可生效。
- 如需更换端口，点击「端口检测与更换」按钮，工具会自动检测占用情况并推荐可用端口。

### 备份与还原
- 在「世界管理」标签页中点击「立即备份当前世界」，程序会将世界压缩为 ZIP 并保存到 `backups/`。
- 选择已有备份，点击「还原备份」即可覆盖当前世界（需服务器停止）。

### 隧道穿透
1. 在「隧道」标签页中设置 `frpc.exe` 路径（从 ChmlFrp 官网下载 https://www.chmlfrp.net/ ）。
2. 编辑 `frpc.ini` 配置内容（可粘贴从官网生成的配置）。
3. 点击「启动隧道」即可连接内网穿透服务。

### 版本升级
1. 点击「检查稳定版更新」或「检查预览版更新」，程序会自动探测最新版本号。
2. 如有新版本，点击「下载最新版本」选择保存路径。
3. 下载完成后，点击「开始升级」即可自动备份并覆盖核心文件（需服务器停止）。
